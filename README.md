### 과제 후기

이번 과제를 통해 Next.js를 처음으로 실제 요구사항에 맞춰 사용해볼 수 있었습니다. 지금은 단일 페이지 구조라 폴더 기반 라우팅에 고민이 크지 않았으나, 화면이 복잡한 구조라면 설계와 역할 분리에 대한 고민이 깊어질 것 같습니다.  
또한 모바일, PC 반응형 UI 역시 처음으로 구현해봤습니다. 빠른 완성을 목표로 가장 익숙한 Tailwind CSS를 사용해 구현했지만, 다른 대안에 대해서도 확인해보고 싶다는 생각이 들었습니다.  
좋은 과제를 통해 새로운 경험을 할 수 있었습니다. 감사합니다.
